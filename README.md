This project implements vehicle detection on a video file (`younus.mp4`) and live using **YOLOv4 with MobileNetV2 backbone** and **OpenCV**. The system detects vehicles frame by frame and saves the annotated output video.

---

## Features
- Detects vehicles from a video file
- Uses YOLOv4 MobileNetV2 backbone for speed and accuracy
- Saves output as `output_video.mp4`
- Displays detection in real-time

---

## Requirements
- Python 3.6+
- OpenCV
- NumPy
